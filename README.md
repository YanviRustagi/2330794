# Campus Notification System

## Stage 1

* Priority Inbox implementation
* Notification sorting based on priority and recency
* Top N notifications display

## Stage 2

* Next.js frontend
* Material UI components
* Notification cards
* Filter tabs (All, Placement, Result, Event)

## Run Frontend

```bash
npm install
npm run dev
```

Open:

http://localhost:3000

## Technologies Used

* Next.js
* TypeScript
* Material UI
* JavaScript
